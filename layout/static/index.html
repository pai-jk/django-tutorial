Doctris - Doctor appointment booking system

Introduction

Doctris is a clean, modern, creative & responsive HTML template and Admin Dashboard for appointment booking system. It is built with Official latest Bootstrap 5.0.2 and Sass. This can be used for any health-related website such as doctors clinic, hospital, doctor consultation center, cosmetic surgery center, medical laboratory, dental clinic, centenary clinic, pharmacy, etc. Fully responsive design layouts with visitor-centric stunning look and feel will help reach the business’s target audience.

We have added a Dark version with RTL supported in both Light and Dark version.

Please feel free to get back to me in case if you are having any question or feedback.

CSS

Files are explained below:

File Description
bootstrap.min.css Doctris uses the latest bootstrap v5.0.2. The core bootstrap file is being used in all the pages.
style.css This file is containing all common styles for all the pages.
style-rtl.css In order to have RTL mode enabled, replace the reference of style.css to style-rtl.css
style-dark.css In order to have Dark mode enabled, replace the reference of style.css to style-dark.css
style-dark-rtl.css In order to have Dark RTL mode enabled, replace the reference of style.css to style-dark-rtl.css
Javascript

Files are explained below:

File Description
bootstrap.bundle.min.js, etc. These files are used at core of the theme.
app.js This is a main js file. It contains the custom JS code needed for features including layout, sidebar, etc.
Loader

       <div class="double-bounce1"></div>
       <div class="double-bounce2"></div>
     </div>
   </div>
</div>
<-- loader end -->
